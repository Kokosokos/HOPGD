Detailed Description

N-dimensional matrix(tensor) container class.

Considers first 2 dimensions as largest (space & time) and all the rest as additional parameters. Stores N-dimensional matrix as a vector of 2D matrices, but can return an N-dimensional index.

intVector NDMatrix::m_cumul_sizes

The vector containing the cumulative sizes of each dimension.

m_cumul_sizes[0]=m_sizes[0], m_cumul_sizes[n]=m_cumul_sizes[n-1]*m_sizes[n]. Used to speed-up the ND index calculation.
